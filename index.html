This project is maintained by meritt

View helpers for any handlebars projects. For express 3 I prefer to use express-hbs and for koa I prefer to use koa-hbs. Btw, all helpers can work without handlebars.

Installation

$ npm install hbs-helpers

Configure for express-hbs

var hbs = require('express-hbs');
var helpers = require('hbs-helpers');

for (var helper in helpers) {
  hbs.registerHelper(helper, helpers[helper]);
}

Helpers

  • downsize (text[, options])
  • length (number)
  • formatdate (date[, options])
  • rusweek (date[, options])
  • formatyear (date)
  • timeago (date[, options])
  • formatmoney (number)
  • plural (count[, options])
  • rubles (number)
  • htmlescape (text)
  • urlencode (text)
  • striptags (text)
